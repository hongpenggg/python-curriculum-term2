# Bit by Bit Coding — Term 2 Python Curriculum

**Python: Language & Application**  
Term 2 — July to October 2026

## Overview

This repository contains the complete curriculum for Bit by Bit Coding's Term 2 Python course. The term runs for 16 weeks, beginning with 4 Taster sessions (open to new students) followed by 12 weeks of structured learning, a 3-week hackathon, and a final data representation session.

## Folder Structure

| Folder | Topic | Date |
|--------|-------|------|
| `Taster 1 - Python Variables` | Icebreakers, Bonding, Intro to Coding, Python Variables | 18 Jul 2026 |
| `Taster 2 - Control Flow` | Comparison Operators, Logical Operators, if/elif/else | 25 Jul 2026 |
| `Taster 3 - Loops` | while loops, for loops, range(), break, continue | 1 Aug 2026 |
| `Taster 4 - Application Session` | Combining all taster skills in a guided mini-project | 8 Aug 2026 |
| `Week 1 - Lists, Dictionaries, Tuples` | Data structures, list/dict operations, comprehensions | 15 Aug 2026 |
| `Week 2 - Functions, Scope` | def, parameters, return values, scope, *args/**kwargs | 22 Aug 2026 |
| `Week 3 - File IO` | Reading/writing files, CSV processing, text analysis | 29 Aug 2026 |
| `Week 4 - Recursion` | Base case, recursive case, factorial, fibonacci | 5 Sep 2026 |
| `Week 5 - Algorithms I (Searching)` | Linear search, binary search, time complexity | 12 Sep 2026 |
| `Week 6 - Algorithms II (Sorting)` | Bubble sort, selection sort, insertion sort | 19 Sep 2026 |
| `Week 7 - OOP I` | Classes, objects, __init__, self, attributes, methods | 26 Sep 2026 |
| `Week 8 - OOP II & Data Structures` | Inheritance, linked lists, binary trees, stacks, queues | 3 Oct 2026 |
| `Hackathon (Weeks 9-11)` | "Code for Tomorrow" Grand Challenge | 10-24 Oct 2026 |
| `Week 12 - Data Representation & APIs` | Binary, hexadecimal, ASCII, APIs, JSON | 31 Oct 2026 |

## Each Folder Contains

- **Demo.ipynb** — Live coding walkthrough notebook (tutor-led)
- **Challenges.ipynb** — Micro-challenges and collaborative group challenge
- **Extra Practice.ipynb** — Additional practice problems for self-study
- **Answers/** — Answer notebooks for tutors
- **Cheatsheet.pdf** — Quick reference for key concepts
- **Lesson Plan.docx** — (Taster sessions only) Detailed lesson plan with timing

## Session Structure (3 hours)

| Time | Segment |
|------|---------|
| 0:00 - 0:15 | Attendance, recap, Q&A |
| 0:15 - 1:15 | New concept introduction (live coding demo) |
| 1:15 - 1:30 | Break |
| 1:30 - 2:00 | Hands-on practice (micro-challenges) |
| 2:00 - 2:50 | Pair programming / group activity |
| 2:50 - 3:00 | Wrap-up, preview, homework |

## Age Differentiation

- **Ages 10-12 (Gamifiers):** Turtle Graphics, Pygame, physical analogies for complex concepts
- **Ages 13-15 (Problem Solvers):** CLI applications, O-Level Computing tier problems
- **Ages 16-18 (Optimizers):** Efficiency, clean code, algorithm complexity analysis

## Hackathon: Code for Tomorrow

The Term 2 Grand Challenge runs over 3 weeks (Weeks 9-11). Students build a Python application that works with data to solve a real problem. See `Hackathon (Weeks 9-11)/BbB_GrandChallenge_Term2_2026.docx` for full details.
